
## pip 包管理

常用的 pip 命令：
```
pip3 install package_name    //安装指定的 Python 包。
pip3 uninstall package_name  //卸载指定的 Python 包。
pip3 freeze                  //列出当前环境中安装的所有包及其版本。
pip3 list                    //列出当前环境中安装的所有包。
pip3 show package_name       //显示指定包的详细信息，包括版本、作者等。
pip3 search package_name     //在 PyPI 中搜索指定的包。
```
